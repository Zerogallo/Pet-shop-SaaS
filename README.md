# 🐾 PetShop SaaS – Agendamento + Prontuário + Gestão de Produtos

Sistema completo para gestão de pet shops, desenvolvido como projeto fullstack para um cliente. Oferece **agendamento online**, **prontuário eletrônico dos pets** e **controle de produtos/estoque**, tudo em uma interface moderna e responsiva.

## ✨ Funcionalidades principais

- 📅 **Agendamentos online** – data, hora, serviço, vínculo com o pet
- 🐕 **Prontuário eletrônico** – histórico médico, prescrições, veterinário responsável
- 📦 **Gestão de produtos** – cadastro, preço, estoque, categorias
- 👥 **Usuários** – autenticação JWT (login/registro)
- 📊 **Dashboard** – visão rápida dos últimos pets, produtos e próximos atendimentos
- 🎨 **UI responsiva** – funciona no desktop e dispositivos móveis

## 🛠️ Tecnologias utilizadas

### Frontend
- React 18
- Vite
- CSS puro (sem frameworks externos)
- Axios
- React Router DOM

### Backend
- Node.js
- NestJS
- TypeORM
- SQLite (pode ser trocado por PostgreSQL)
- JWT (autenticação)
- Bcrypt

## 🚀 Como executar o projeto

### Pré‑requisitos

- Node.js (versão 18 ou superior)
- npm (ou yarn)

### Backend

```bash
cd backend
npm install
npm run start:dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
